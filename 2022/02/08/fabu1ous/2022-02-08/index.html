[하루한줄] HigherLogic Community RCE Vulnerability

URL

HigherLogic Community RCE Vulnerability

Target

HigherLogic Community

Explain

__ViewState는 사용자 세션의 상태에 대한 정보를 포함하는 매개 변수입니다.

.NET에서는 사용자가 로그인되어 있는지, 사용자 설정이 변경되었는지 등의 정보가 serialized 된 .NET object의 형태로 __ViewState 매개 변수 안에 있습니다. 사용자는 웹서버에 요청을 보낼 때 웹 사이트에서 제공하는 __ViewState를 전송하고 서버를 이를 사용해 사용자 세선을 관리합니다.

2000년대 후반, .NET을 사용하는 많은 웹 사이트 개발자들은 __ViewState 대신 이를 압축한 __VState에 사용자 뷰스테이트를 저장하는 코드를 추가하여 사용했습니다. 문제는 __ViewState에서 deserialization에 의한 원격 코드 실행 취약점이 발견되었고 이를 방지하기 위해 암호화 설정이 추가되었는데, __VState 트릭을 사용하는 애플리케이션들은 개발자가 직접 수정하지 않는 한 이러한 업데이트를 받지 못합니다.

즉, __VState를 사용하는 .NET 웹 애플리케이션들 중 다수가 deserialization에 의한 원격 코드 실행 취약점이 그대로 남아 있어 악용이 가능합니다.
